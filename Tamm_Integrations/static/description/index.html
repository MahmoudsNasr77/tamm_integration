Tamm Fleet Management Integration (Saudi Arabia)

A Complete Solution for Real-Time Fleet Tracking, Maintenance, Fuel Management, and Operational Analytics

Core Features

  • Real-time GPS vehicle tracking with interactive map view
  • Comprehensive maintenance planning and service alert automation
  • Fuel usage monitoring and cost analysis tools
  • Driver behavior evaluation and safety scoring
  • Optimized route planning for enhanced fleet efficiency
  • Security alerts, notifications, and fleet movement insights
  • Advanced dashboards and KPI-based reporting
  • Support for multi-company fleet operations
  • Fully aligned with the Saudi Arabian TAMM Fleet System

Installation Guide

1. Copy the Tamm_Integrations module folder into your Odoo addons directory.
2. Enable Developer Mode and update the Apps list.
3. Install the module titled "Tamm Fleet Management Integration".
4. Navigate to Tamm Fleet : Configuration : Settings.
5. Enter your TAMM API Key, Secret, and Base URL provided by the official TAMM service, then select Test Connection.
6. Use the Tamm Dashboard to monitor vehicles, alerts, and operational data in real time.
