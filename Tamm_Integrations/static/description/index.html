Installation & Documentation

Follow these steps to install and configure:

  1. Copy the Tamm_Integrations module folder into your addons directory.
  2. Enable Developer Mode → Update Apps list.
  3. Install the module titled "Tamm Fleet Management Integration".
  4. Navigate to Tamm Fleet → Configuration → Settings.
  5. Enter your TAMM API Key, Secret, and Base URL then click Test Connection.
  6. Access the Tamm Dashboard to monitor your fleet.
